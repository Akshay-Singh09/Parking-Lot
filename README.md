# Design a Parking Lot System

### Functional Requirements

- Support different vehicle types (Two-wheeler, Four-wheeler)
- Support different payment methods (CASH, CARD, UPI)
- Generate parking tickets with unique IDs
- Track real-time parking availability
- Handle multiple parking floors and spots
- Provide parking spot search by vehicle type
- Calculate parking fees based on duration and vehicle type


### Entity

- Vehicle
- Ticket
- Bill
- ParkingLot
- Payment
